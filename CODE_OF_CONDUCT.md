# Code Of Conduct

Session Portal is intended to be a useful, respectful open-source project for developers working with AI CLI tools.

## Expected Behavior

- Be respectful and direct.
- Keep issues and pull requests focused on the project.
- Assume good intent, but verify technical claims with evidence.
- Avoid sharing private logs, secrets, local paths, or session contents.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Posting private data from another person or organization.
- Spamming issues, pull requests, or discussions.
- Using the project space to distribute malicious code or instructions.

## Reporting

If there is a conduct or safety concern, open a private security report when appropriate or contact the maintainer through GitHub.
